# k-means-cluster

I applied the K-Means Clustering algorithm to the Mall_customers.csv dataset. The goal is to uncover hidden patterns in customer spending habits. By calculating the Within-Cluster Sum of Squares (WCSS), I determined the optimal number of clusters to effectively segment the mall's client base for targeted marketing strategies.
